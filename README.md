# Family Budget Calculator

HTML/JS калькулятор семейного бюджета с авторизацией по email, хранением данных в `localStorage`, аналитикой, календарём и графиками.

## Что доработано

- до входа в аккаунт приложение запускается в пустом состоянии;
- без авторизации все показатели калькулятора остаются нулевыми;
- демонстрационные данные на старте удалены;
- после выхода из аккаунта интерфейс очищается и снова показывает нули;
- структура проекта приведена к виду небольшого полноценного репозитория.

## Локальный запуск

```bash
npm run dev
```

После этого откройте локальный адрес из консоли.

## Поведение авторизации

Приложение хранит пользовательские данные отдельно по ключу:

```text
familyBudgetFinalV1::<email>
```

Пока пользователь не вошёл:
- данные не загружаются;
- автосохранение отключено;
- показатели интерфейса остаются нулевыми.

Это корректный сценарий для клиентского MVP, если вы хотите скрывать финансовые данные до авторизации. Но это не является настоящей защитой данных на уровне сервера, потому что текущий проект целиком работает в браузере и использует `localStorage`.
